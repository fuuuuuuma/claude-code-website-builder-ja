# 🎁 画像だけ作らせる（GPT Image 2.0）

背景・装飾・OG画像などをAIに1枚生成させるプロンプト。

**受け取り方**: 下のコードブロック右上のコピーボタンでコピーするか、このファイルをそのままダウンロードしてください。

**使い方**: ヒーローや装飾に画像が要るときに送る。

```text
OPENAI_API_KEY を設定して、scripts/gen_image.py（gpt-image-2）でヒーロー背景を1枚作って（横長・◯◯な雰囲気・文字は入れない）。
重要な見出し・価格・CTAは画像に焼き込まず、HTMLテキストで置いて。生成後に崩れを目視して。
```

**ポイント**: OpenAI APIのgpt-image-2を使う（OPENAI_API_KEYが必要）。重要テキストは必ずHTMLに。
